Projet Méthodo tests/tests unitaires RAMDANI Rayan, ABID Sabri et HAROUNE Rayane

Dans ce projet nous avons une WebApp avec plusieurs bugs.
Notre objectif et de réaliser une série de tests afin de corriger toutes ces anomalies.
Avant de pouvoir commencer les tests, faut d'abord paramétrer sa machine en installant les extensions suivantes : 

click==7.1.2
Flask==1.1.2
itsdangerous==1.1.0
Jinja2==2.11.2
MarkupSafe==1.1.1
Werkzeug==1.0.1
pytest==6.2.5 

et ce en exécutant dans le CMD la commande pip install suivis du nom de l'extension ex: " pip install pytest "
